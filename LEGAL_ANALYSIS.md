# Legal Analysis: GPLv3 for the BOSC Community Library

**Course:** BSCT 3221 - Open Source Software  
**Project:** BOSC Community Library  
**Chosen License:** GNU General Public License version 3.0 (GPLv3)

## Introduction

For this project I chose the GNU General Public License version 3.0 because the BOSC Community Library is meant for education and public benefit. A digital library for schools, universities, and community learning centers should not depend on closed rules that only one vendor controls. The users should be able to inspect the software, improve it, share it, and keep using it even if a service provider changes prices or stops supporting them.

## Why GPLv3 Fits A Public-Sector Project

GPLv3 is a strong choice for a public-sector project because it protects the software as a shared resource. If government institutions, students, lecturers, or local developers help improve the BOSC Community Library, those improvements should not disappear into a closed commercial product. A permissive license such as MIT or BSD would allow a company to take the code, modify it, and sell a closed version without giving the improvements back.

That may be acceptable for some projects, but I think it is risky for education software. Schools often have limited budgets, and many public institutions already struggle with expensive subscriptions. GPLv3 reduces that risk through copyleft. If someone distributes a modified version of the system, they must also provide the source code under the same license. This keeps improvements available to the community.

The license also supports transparency. Public institutions need software that can be audited for security, privacy, procurement fairness, and long-term maintainability. With GPLv3, technical teams are allowed to read the code and adapt it to local needs. For example, if a school wants Luganda support, offline access, or a different resource catalog structure, the license allows that work to happen without asking permission from a private vendor.

## Patent And Trademark Issues

GPLv3 also handles patent risk better than very simple licenses. It includes a patent grant from contributors, which helps protect users from patent claims connected to contributed code. This matters because a school or public office should not adopt a system and later discover that a contributor is trying to use patents against users of the software.

Compared with Apache License 2.0, GPLv3 is stricter because it combines patent protection with copyleft. Apache 2.0 also has patent language, but it allows proprietary redistribution. GPLv3 is better for this project because it makes sure that distributed improvements remain open.

Trademark protection is different. GPLv3 allows people to copy and modify the code, but it does not automatically give them the right to misuse the BOSC name or pretend that an unofficial version is the official project. As the project grows, it would be important to add simple branding guidance so that forks can exist without confusing schools and users.

## Commercial Use And Paid Versions

GPLv3 does not stop people from making money. A company can offer paid hosting, installation, training, backups, security updates, customization, and support. This is actually useful, because many schools may not have full-time technical staff. What GPLv3 prevents is taking the community version, closing the source code, and locking users into one vendor.

A commercial provider could therefore build a paid service around BOSC Community Library, but if it distributes modified software, it must share the modified source code. This creates a fair balance. Businesses can earn from services, while schools keep the freedom to move to another provider, self-host, or ask local developers to continue the work.

## Conclusion

GPLv3 fits the BOSC Community Library because it protects public access, transparency, and long-term reuse. It allows commercial participation, but it does not allow public-interest work to be quietly converted into closed infrastructure. For an education platform intended to serve communities, that protection is important.
