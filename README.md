# BOSC Community Library 📚

Welcome to the **BOSC Community Library**, an open-source digital library system designed to provide accessible educational resources. This project is built with a focus on public-sector adoption, aiming to support schools, universities, and government institutions in Uganda and beyond.

## 🌟 Features
- **Resource Management:** Organize and distribute educational materials efficiently.
- **Open Standards:** Built for transparency, sustainability, and easy auditing.
- **Localization-Ready:** Designed to support multiple languages, including Luganda.
- **Laravel Framework:** Leveraging PHP and Laravel for robust, secure backend services.

## 🚀 Getting Started

To run this project locally, ensure you have PHP, Composer, and Node.js installed.

```bash
# Clone the repository
git clone https://github.com/ThrustSoftwares/BOSC-Community-Library.git

# Navigate into the directory
cd BOSC-Community-Library

# Install PHP dependencies
composer install

# Install JS dependencies
npm install

# Copy environment variables
cp .env.example .env

# Generate application key
php artisan key:generate

# Run the local server
php artisan serve
```
